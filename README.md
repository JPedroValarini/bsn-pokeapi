## 📌 Sobre o Projeto
API RESTful inspirada na PokéAPI para consulta de Pokémon, desenvolvida como desafio técnico.

### 🎯 Funcionalidades
- Listagem de Pokémon (com paginação)
- Busca por ID/Nome
- Filtros por Tipo/Habilidade

## 🛠️ Tecnologias
- PHP | Laravel | MySQL
